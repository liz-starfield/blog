## Github上markdown特点

- 链接引用不能有空格 <br/>
  - 解决：markdown文件名不要有空格 <br/>
- 不支持[TOC]生成标题目录 <br/>
  - 解决：在VScode中安装插件Markdown All in One，打开md文件，使用Ctrl+Shift+P快捷键，输入命令Markdown All in One: Create Table of Contents，回车，即在md文件中加上目录
